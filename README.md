# Modding By Kaupenjoe: FABRIC MODDING COURSE 26.X

This is the GitHub Repository for Kaupenjoe's Minecraft Modding Course using Fabric for Minecraft version 26.X.

It covers a variety of Topics: From Minecraft Modding Basics, to Custom Tools, Armor, Dynamic World Generation, over Custom Mobs, Dimensions and Block Entities and much more! The Source Code will always be freely available under the MIT-License, but to support me you can buy the Course which gets you 20+ Hours of Video Lectures explaining the code in detail!
